Umbrella Today?

Ruby, Ruby on Rails, Javascript, jQuery, Handlebars, Bootstrap

View site
GitHub front end
GitHub back end

Full stack web app that uses DarkSky's API to determine if you need an umbrella today.

Forecasts can be searched by location name or coordinates. Locations searchs by name utilizes forward geocoding via geocode.xyz's API.

Registered users have the ability to save locations and see them displayed as a list.
